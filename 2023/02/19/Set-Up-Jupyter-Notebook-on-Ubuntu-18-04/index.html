Set Up Jupyter Notebook on Ubuntu 18.04

Posted on 2023-02-19 In Tutorials , Jupyter Notebook Views: Valine:
Symbols count in article: 4.8k Reading time ≈ 4 mins.

Jupyter Notebook is an open-source web application that lets you create and share interactive code, visualizations, and more. This tool can be used with several programming languages, including Python, Julia, R, Haskell, and Ruby. It is often used for working with data, statistical modeling, and machine learning.

This tutorial will walk you through setting up Jupyter Notebook to run from an Ubuntu 18.04 server, as well as teach you how to connect to and use the Notebook. Jupyter Notebooks (or simply Notebooks) are documents produced by the Jupyter Notebook app which contain both computer code and rich text elements (paragraph, equations, figures, links, etc.) which aid in presenting and sharing reproducible research. In the final step of this guide, you will run Python 3 code using a Jupyter Notebook running on a remote server.

Installation

Please make sure python3 and pip3 are installed. If not, install them using the following command.

pip install notebook

jupyter –version

from notebook.auth import passwd;
passwd()

jupyter notebook –generate-config

vi .jupyter/jupyter_notebook_config.py

Whether to allow the user to run the notebook as root. If you do not install jupyter in a virtual env, please set this option as True.

c.NotebookApp.allow_root = True

The IP address the notebook server will listen on. ‘*’ means any IP.

c.NotebookApp.ip = ‘*’

The directory to use for notebooks and kernels. Please do not use full path, jupyter will add the default path prefix automatically.

c.NotebookApp.notebook_dir = ‘gits/jupyter/‘

Whether to open in a browser after starting. The specific browser used is

platform dependent and determined by the python standard library webbrowser

module, unless it is overridden using the –browser (NotebookApp.browser)

configuration option.

c.NotebookApp.open_browser = False

The string should be of the form type:salt:hashed-password. Copy the token generated by passwd().

c.NotebookApp.password = ‘argon2:$argon2id$v=19$m=10240,t=10,p=8$x/wZxj7yQsoUagjuidgnaiudgiu9adniaunaneianngda’

The port the notebook server will listen on (env: JUPYTER_PORT).

c.NotebookApp.port = 8888

jupyter notebook

sudo ufw allow 8888

sudo vi /lib/systemd/system/jupyter.service
